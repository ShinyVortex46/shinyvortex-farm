# ShinyVortex Farm

A personal shiny hunting tracker built as a React web app.

## Project concept
A visual "farm" where caught shinies populate habitats.
Hunts are started per species/game/method. Tracks encounter
counts, odds, and statistics across all hunts.

## Tech stack
- React (create-react-app)
- Local JSON storage
- Gen 3 style sprites from PokeAPI
